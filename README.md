# Kross_Lab4

##Создание картинки по номерам.

1. - Для загруженного изображения необходимо произвести кластеризацию для преобразования в режим индексированных (счётного числа) цветов. Например, оставить 5-6 цветов из доступных фломастеров.

2. - Для полученного изображения с однородно заполненными областями (получится похожее на мультипликационный кадр) произвести поиск контуров объектов (областей с разными цветами заливок).

3. - В полученных контурах произвести "отброс" "лишних" точек. То есть применить функционал аппроксимации.

4. - Дальше "проредить" массив точек контура исключая точки, расположенные рядом с друг другом. Нам необходимо потом вывести номера точек и необходимо чтобы номера не накладывались друг на друга. Для прореживания нужно выполнить условие что расстояние между точками больше чем размер надписи номера точек.

5. - Для массива точек контуров запомнить цвет области внутри контура.

6. - Создать новое изображение по размеру совпадающее с исходным, на изображении вывести номера точек и сами точки всех контуров. В центре точек более крупным шрифтом вывести номер контура. Все точки, принадлежащие одному контуру и номер контура необходимо выводить цветом, сохраненным в п. 5.

7. - (extra) Номера точек необходимо позиционировать так, чтобы в закрашенной контуре они были внутри закрашенного контура. Следующая мысль неверна:(По сути здесь необходимо найти геометрический центр контура и располагать надпись на линии между центром и точкой. Все точки, которые находятся справа от центра контура необходимо выводить с выравниванием по правому краю, все что слева - с выравниванием по левому краю).  Правильней располагать точки справа от движения по пути обхода точек на биссектрисе угла центральной среди трех обходимых точек.
